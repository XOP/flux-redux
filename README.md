# flux-redux

> Redux data flow example


## Install

```
npm install
```


## Commands

**Build**
```
npm run build
```

**Dev mode**
```
npm run dev
```

**Run tests**
```
npm test
```


## Notes

[Babel](https://babeljs.io/docs/usage/cli/)(babel-node) is used only for tests compilation, so config is set in [package.json](package.json) for brevity and consistency.
